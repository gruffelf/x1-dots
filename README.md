# x1-dots
#### Atom One Dark Color Scheme
### Applications Required
WM - SwayWM  
Status bar - swayrbar or waybar (configs for each provided)  
Term - Foot  
Wallpaper - swaybg (wallpapers from [https://github.com/Narmis-E/onedark-wallpapers]) 
App Menu - fuzzel  
### Additional Applications
nvim  
btop  
spotify_player
Firefox - One dark theme
